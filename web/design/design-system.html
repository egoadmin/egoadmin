PROTO FIRST · HTTP + gRPC

少写后台基础设施，
把业务系统快速跑起来

EgoAdmin 不是只带登录和 CRUD 的后台模板，而是一套面向 Go 微服务的开发底座 —— 用 Proto First 串起接口、验证、文档、权限和响应映射。

role.proto半个接口工程
// 定义 proto，就完成半个接口工程
service RoleService {
  rpc AddRole(AddRoleRequest)
      returns (AddRoleResponse);
}
message AddRoleRequest { string name = 1; }
→ gRPC / HTTP · 校验 · OpenAPI · 权限标识
